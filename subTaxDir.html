How to Fill out Subsequent Taxes CSV File

This CSV file is used to apply the surplus for a property sold at tax sale to taxes that have become due after the sale. Only properties that were sold on the tax sale that generated a surplus will appear on this CSV file. Fill out the form as described below for those properties you want to apply the surplus to subsequent taxes

Any properties left blank will be skipped by the system.
Directions to fill out CSV file
  1. Click 'Download CSV file to fill out' button
  2. Open saved document (check downloads folder)
  3. Fill out columns F-J as per the definitions provided below
  4. Save file locally to your computer
  5. Click 'Choose File' button and select your newly saved CSV file
  6. Click 'Upload CSV file to Process' button
Column Definition
  • Columns A & B: SALE ID, PROPERTY ID - Are to identify the property
  • Columns C, D, & E: SURPLUS, SURPLUS WARRANTS, SURPLUS REMAINING show the amount of the surplus, amount of warrants already paid, and how much surplus remains
  • Columns F, G, H, I, & J: Need to complete as described below
Field Definitions
  • DATE (Column F) - Put in the date of the warrant in the format mm/dd/yyyy (EX: 9/11/2021)
  • NUMBER (Column G) - Put in the number of the warrant
  • PAID TO (Column H) - Put in to whom this warrant will be paid to
  • AMOUNT (Column I) - Put in the amount of the warrant without a '$' or commas (EX: $2,123.99 would be entered as 2123.99)
  • DESCRIPTION (Column J) - Put in the description for the offset adjustment that will also be entered (EX: Used Surplus for 2021 / Pay 2022)
