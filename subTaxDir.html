How to Fill out Subsequent Taxes CSV File

This CSV file is used to apply the surplus for a property sold at tax sale to taxes that have become due after the sale. Only properties that were sold on the tax sale that generated a surplus will appear on this CSV file. Fill out the form as described below for those properties you want to apply the surplus to subsequent taxes.

Directions to fill out CSV file
  1. Click 'Download CSV File file to fill out' button
Column Definition
  • Columns A & B: SALE ID, PROPERTY ID - are to identify the property.
  • Columns C, D, & E: SURPLUS, SURPLUS WARRANTS, SURPLUS REMAINING show the amount of the surplus, amount of warrants already paid, and how much surplus remains
  • Columns F, G, H, I, J, & K: Need to complete as described below.
Field Definitions
  • REDEEM/SURPLUS - Put in surplus if a surplus warrant OR redeem if this is a redemption warrant
  • DATE - Put in the date of the warrant in the format mm/dd/yyyy (EX: 9/11/2021)
  • NUMBER - Put in the number of the warrant
  • PAID TO - Put in to whom this warrant will be paid to
  • AMOUNT - Put in the amount of the warrant without a '$' or commas (EX: $2,123.99 would be entered as 2123.99)
  • DESCRIPTION - Put in the description for the offset adjustment that will also be entered
